Changelog

Changelog atau catatan perubahan dalam berbagai versi yang telah dirilis pada platform Android dan iOS.

Versi Android.

2.9.1 (11 Juli 2022)

  • Penambahan fitur pengaturan lokasi otomatis dan manual untuk lokasi Prakiraan Cuaca
  • Penambahan informasi Gempabumi Real-time
  • Pemutakhiran informasi Peringatan Dini Cuaca dengan penambahan detail gambar peta
  • Pemutakhiran informasi Gempabumi dengan penambahan peta guncangan (shakemap)
  • Perbaikan beberapa bug

    • 2.9 (29 April 2022)

  • Penambahan informasi Indeks Sinar Ultraviolet (UV)
  • Penambahan informasi Cuaca Berbasis Dampak
  • Penambahan tautan eksternal untuk Cuaca Event
  • Pemutakhiran informasi Kualitas Udara
  • Pemutakhiran informasi Cuaca Karhutla
  • Perbaikan beberapa bug

    • 2.8.1 (21 Desember 2020)

  • Penambahan slogan: Multi-Hazard Early Warning System

    • 2.8 (25 September 2020)

  • Tersedia dalam format dua bahasa, Bahasa Indonesia dan Bahasa Inggris
  • Penambahan informasi Konsentrasi PM2.5 pada menu Kualitas Udara
  • Optimasi aplikasi
  • Perbaikan beberapa bug

    • 2.7 (23 Maret 2020)

  • Penambahan informasi prakiraan cuaca dalam 7 harian di seluruh kecamatan di Indonesia
  • Penambahan informasi ada atau tidaknya potensi tsunami pada daftar Gempabumi M ≥ 5.0
  • Penambahan informasi pada Glosari Cuaca
  • Perbaikan beberapa bug

    • 2.6 (23 November 2019)

  • Penambahan area Cuaca Maritim Wilayah Perairan di Indonesia
  • Penambahan informasi Sistem Aplikasi Cuaca untuk Kebakaran Hutan dan Lahan (Sancakarla) yang terdiri atas: GeoHotspot, Citra Sebaran Asap, dan Potensi Karhutla dalam jangka waktu harian dan bulanan
  • Perbaikan beberapa bug

    • 2.5 (1 Oktober 2019)

  • Penambahan pengaturan lokasi cuaca untuk kota/kabupaten
  • Penambahan pengaturan notifikasi peringatan dini cuaca untuk provinsi
  • Penambahan sesar/patahan pada peta gempabumi
  • Perubahan legenda kualitas udara PM10
  • Perbaikan tampilan peta dasar
  • Perbaikan beberapa bug

    • 2.4.1 (4 Oktober 2018)

  • Penambahan informasi Cuaca Event (Asian Para Games 2018 dan The 2018 IMF-WBG Annual Meetings Bali)
  • Perbaikan tampilan peta dasar

    • 2.4 (17 Agustus 2018)

  • Penambahan informasi Cuaca Event (Asian Games 2018)
  • Penambahan informasi Peta Iklim (Hari Tanpa Hujan, Prakiraan dan Analisis Hujan Bulanan)
  • Penambahan informasi Titik Panas (GeoHotspot)
  • Penambahan fitur share ke media sosial
  • Optimasi aplikasi
  • Perbaikan beberapa bug

    • 2.2 (16 Juni 2017)

  • Penambahan fitur NOTIFIKASI untuk Gempabumi, Peringatan Dini Cuaca, dan Press Release / Info Aktual
  • Penambahan informasi Cuaca Jalur Mudik Lebaran disertai trafik lalu lintas jalan (hanya ditampilkan ketika momen mudik)
  • Penambahan informasi Cuaca Bandara di seluruh wilayah Indonesia
  • Optimasi aplikasi
  • Perbaikan beberapa bug

    • 2.1 (12 Februari 2017)

  • Perubahan beberapa keterangan pada Cuaca
  • Perubahan waktu munculnya Prakiraan Cuaca
  • Penambahan beberapa kota untuk lokasi Kualitas Udara (Konsentrasi PM10) disertai status alat
  • Update Gempabumi Terkini mencakup gempabumi dirasakan terkini (sebelumnya hanya gempabumi M ≥ 5.0 terkini)
  • Penambahan animasi lokasi titik gempabumi dan perubahan peta
